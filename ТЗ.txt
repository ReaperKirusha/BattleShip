два игрока

4 поля 9х9 , корабли игрока , выстрелы
значения поля для кораблей: пусто, подбит, стоит, пустое попадание
значения поля для выстрелов попал, непопал, ореол

1 по 4, 2 по 3, 3 по 2, 4 по 1
в соседних клетках нельзя ставить корабли

выстрелы каждый ход

если попал по кораблю то след ход того игрока, кто попал

ореол - область вокруг корабля размером 1 клетка, там нельзя ставить другие корабли

	У каждаго игрока: 
		2 поля 9х9:
			значения поля для кораблей: пусто(O), подбит(D), стоит(S), пустое попадание(M)
			значения поля для выстрелов попал(D), непопал(M), ореол(A), Пусто(O)
		
		корабли с соотвествующим колвом элементов, на каждом элементе значение подбит\не подбит	
		
		сколько осталось кораблей

	Номер хода
	
	чей сейчас ход
	
	

Игра:
	предигровая фаза:
		создается поле, 2 игрока, и две матрицы под размещение

		1 игрок ставит корабли:
			выбрать тип корабля из оставшихся или убрать уже стоящий

			дальше решить поставить его или выбрать другой корабль

			выбрать начальную и конечную точку куда ставится корабль, (для корбля 1х1 конечная точка задается автоматически)
		
			виды ошибок при расстановке:
				выбраны точки за пределами матрицы
				точки не в одном столбце или строке
				длина последовательности больше или меньше длины корабля
				точки стоят на ореоле другого корабля

		2 игрок тоже самое

		копирование матрицы растановки в матрицу корабелй

	игровая фаза:
		случайно выбрать игрока, который ходит первым, условно игрок Х1 и Х2
		
		подтвердить что ходит игрок Х1
		отрисовать поле для него
		далее ход Х1:
			выбрать точку выстрела
			виды ошибок:
				точка за пределами поля
				выбрана уже ранее выбиравшаяся точка или ореол
			
			далее узнать что с выстрелом
			3 исхода:
				1) попал:
					записать в поле для выстрелов Х1:
						попал, расставить ореолы
						
					записать попадание в соотв. корабль для Х2	
			
					записать в поле для кораблей Х2:
						подбит
				2) непопал:
					записать в поле для выстрелов Х1:
						непопал	
			
					записать в поле для кораблей Х2:
						пустое попадание
				3) уничтожил корабль:
					записать в поле для выстрелов Х1:
						попал, расставить ореолы	

					записать попадание в соотв. корабль для Х2
					уменьшить кол-во кораблей на 1 у Х2
					
			
					записать в поле для кораблей Х2:
						подбит
			
			Конец хода:
				вывести новую информацию о поле и сообщение о исходи выстрела
				в случае победы написать победа игрока Х1
				в ином случае спросить о завершении хода

		подтвердить что ходит игрок Х1
		отрисовать поле для него
		далее ход Х2:
			то же что и для Х1
